# mp3master
воспроизведение большой музыкальной коллекции различными способами
18:38 29.12.2016
Программа для управления воспроизведением музыки ,хранящейся
в различных каталогах. Она использует готовые консольные проигрыватели
типа mpg123. Проверена в системе WindowsXP. Написана на языке Racket5.3
Вся необходимая информация хранится в файле upr.conf ,на первой строке
стоит название текстового файла со списком директорий ,на второй путь
к проигрывателю : "klassika.txt"
                  "c:/mpg123/mpg123.exe"
       
    Текстовых файлов со списком директорий может быть сколько угодно много,но в upr.conf записывается только один. Эти файлы выглядят примерно
    так :"d:/Musik/Mozart/" "d:/Musik/Boka/" "d:/Musik/Vivaldi/"....
    Если папок много ,то составление списка может оказаться очень трудоёмким. Для облегчения этого процесса написаны две небольшие программки:index.rkt и index2.rkt. Они запускаются в консоли ,а в качестве
    аргумента передаётся путь к корневой папке : index.exe "g:/Musik"
index составляет списки для директорий где музыкальные файлы хранятся на
втором уровне: общая папка->папки альбомов->музыкальные файлы. Index2 
применяется там где музыкальные файлы находятся на третьем уровне:
 общая папка->общие папки->папки альбомов->музыка. Из полученных таким
 образом файлов списков необходимо удалить все скобки и только после этого они могут применяться.
    
    Существует три режима воспроизведения : "регулярный" - все папки 
проигрываются в том порядке в каком они записаны;
    "случайный" - папка выбирается случайно ,файлы в папке тоже выбираются
    случайно;
    "случйно-одиночный" - папка выбирается случайно из неё проигрываются два случайно выбранных файла и далее следует переход к другой папке...
    
    Для начала необходимо получить исполнительные файлы : upr.exe ,play.exe ,play-rand.exe ,play-uno.exe. Для этого в консоли набираем :
    
          raco exe <path:>/upr.rkt и так далее...
  
  Потом составляем хотя бы один текстовый файл со списком директорий и
  заполняем файл upr.conf . После этого можно запускать в консоли управляющий модуль upr.exe. Сначала необходимо будет произвести выбор
  режима воспроизведения : "regular" - "1" ; "rand" - "2"; "rand-uno" - "3". Дальнейшее управление осуществляется в консоли и зависит от того
  какой проигрыватель используется. Так для mpg123 надо нажать Ctrl/C 
  один раз ,чтобы пропустить песню и два раза ,чтобы закончить воспроизведение. После исчерпания музыкальной программы произойдёт 

автоматический останов.
